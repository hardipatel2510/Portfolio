# Personal Portfolio

This is a personal portfolio website built with Next.js, React, Tailwind CSS, and ShadCN UI components.

## Getting Started

First, install the dependencies:

```bash
npm install
# or
yarn install
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:9002](http://localhost:9002) with your browser to see the result.
